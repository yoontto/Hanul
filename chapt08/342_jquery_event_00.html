이벤트 : 사이트에서 방문자가 취하는 모든 행위 이벤트 핸들러 : 이벤트가 발생했을 때 실행되는 일련의 실행 코드 이벤트 등록 메서드 1. 단독 이벤트 등록 메서드 : 한 동작에 대한 이벤트를 등록 하나의 이벤트에만 대응하여 이벤트 핸들러에 실행문이 실행 요소를 클릭했을때, 요소를 더블 클릭 했을 때, 요소에 마우스를 올렸을 때 등 버튼을 클릭했을 때 경고창을 띄워라 ☞ 버튼 : 이벤트의 대상, 경고창을 띄워라 : 이벤트 핸들러 2. 그룹 이벤트 등록 메서드 : 한 동작에 대한 이벤트를 2개 이상(여러개) 등록 1. 단독 이벤트 등록 메서드 ★ 이벤트 등록 메서드의 종류 - 로딩 이벤트 : load(), ready() load() : 선택한 이미지 또는 프레임 요소에 연동된 소스가 로딩이 완료된 후 이벤트 발생 ready() : 지정한 HTML 문서 객체가 로딩이 완료된 후 이벤트 발생 - 마우스 이벤트 : click(), dblclick(), mouseover(), mouseout(), hover(), mouseenter(), mouseleave(), mousedown(), mouseup(), mousemove() click() : 선택한 요소를 클릭했을 때 이벤트 발생 dblclick() : 선택한 요소를 더블클릭했을 때 이벤트 발생 jQuery mouseover와 mouseout은 한세트 jQuery mouseenter와 mouseleave가 한세트 mouseover() : 선택한 요소의 영역에서 마우스를 올렸을 때 이벤트 발생, 자식 요소위에 마우스를 over하여도 이벤트가 발생 mouseout() : 선택한 요소의 영역에서 마우스가 벗어날을 때 이벤트 발생, 자식 요소위에 마우스를 out하여도 이벤트가 발생 mouseenter() : 선택한 요소의 영역에서 마우스를 올렸을 때 이벤트 발생, 해당 요소에서만 이벤트가 발생 mouseleave() : 선택한 요소의 영역에서 마우스가 벗어날을 때 이벤트 발생, 해당 요소에서만 이벤트가 발생 hover() : 선택한 요소의 영역에서 마우스를 올렸을 때와 벗어났을 때 이벤트 발생 mousedown() : 선택한 요소의 영역에서 마우스 버튼를 눌렀을 때 이벤트 발생 mouseup() : 선택한 요소의 영역에서 마우스 버튼를 눌렀다 떼었을 때 이벤트 발생 mousemove() : 선택한 요소의 영역에서 마우스를 움직였을 때 이벤트 발생 - 포커스 이벤트(키보드로도 이동 가능시 사용) : focus(), focusin(), focusout(), blur() focus() : 선택한 요소에 포커스가 생성되었을 때 이벤트 발생하거나 또는 선택한 요소에 강제로 포커스를 생성 focusin() : 선택한 요소에 포커스가 생성되었을 때 이벤트 발생 focusout() : 선택한 요소에서 포커스가 다른 요소로 이동되었을 때 이벤트 발생 blur() : 선택한 요소에 포커스가 다른 요소로 이동되었을 때 이벤트를 발생시키거나 선택한 요소에 강제로 포커스가 사라지도록 함 - 키보드 이벤트(키보드로만 이벤트 발생) : keypress(), keydown(), keyup() keypress() : 선택한 요소에서 키보드를 눌렀을때 이벤트가 발생, 그리고 문자키를 제외한 키코드값을 반환 keydown() : 선택한 요소에서 키보드를 눌렀을때 이벤트가 발생, 그리고 자판의 모든 키코드값을 반환 keyup() : 선택한 요소에서 키보드를 떼었때 이벤트가 발생 - 기타 이벤트 : scroll(), change(), error() scroll() : 스크롤바를 움직일때 이벤트 발생 change() : 선택한 요소에 입력요소의 값이 변경되었을 때 이벤트 발생, 이벤트를 강제로 발생시킬 때 error() : 선택한 요소에서 에러가 발생했을때 이벤트 발생 2. 그룹 이벤트 등록 메서드 on() : 선택한 요소에 한개 이상의 이벤트를 등록 (이 중에서는 on만 봐도 됨) bind() : 선택한 요소에 한개 이상의 이벤트를 등록 delegate() : 위임하다, 선택한 요소에 지정한 하위요소에 이벤트를 등록 one() : 선택한 요소에 이벤트를 등록하며, 단 한번만 이벤트가 발생
